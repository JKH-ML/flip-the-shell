개인정보처리방침

Flip the Shell(이하 '확장 프로그램')은 사용자의 개인정보를 소중하게 생각하며, 관련 법령을 준수하기 위해 최선을 다하고 있습니다. 본 방침은 확장 프로그램이 사용자의 정보를 어떻게 처리하는지 설명합니다.

1. 개인정보의 수집 및 이용 목적

본 확장 프로그램은 원칙적으로 사용자의 개인 식별 정보를 수집하거나 외부 서버로 전송하지 않습니다. 모든 데이터 처리는 사용자의 브라우저 내에서 로컬로 수행됩니다.

  • 설정 정보 저장: 사용자가 선택한 아이콘 위치(iconPosition) 등의 설정값을 유지하기 위해 크롬 로컬 저장소(chrome.storage.local)를 사용합니다.

2. 수집하는 개인정보 항목

확장 프로그램은 다음과 같은 비개인적 데이터를 처리합니다.

  • 사용자 설정 값 (예: 아이콘 노출 위치)

이외에 사용자의 웹 서핑 기록, 비밀번호, 쿠키 등 민감한 개인정보는 일절 수집하거나 저장하지 않습니다.

3. 데이터의 제3자 제공 및 위탁

본 확장 프로그램은 수집된 정보를 외부 서버로 전송하지 않으며, 따라서 제3자에게 제공하거나 처리를 위탁하지 않습니다.

4. 권한 사용 안내

기능 수행을 위해 다음과 같은 권한을 사용합니다.

  • storage: 사용자의 설정 값을 로컬 브라우저에 저장하기 위해 사용합니다.
  • activeTab / host_permissions: 웹 페이지 내 이미지를 분석하고 숨겨진 데이터를 해독하기 위해 해당 페이지의 콘텐츠에 접근합니다.
  • scripting: 이미지 해독 로직을 실행하기 위해 사용됩니다.

5. 개인정보의 파기

모든 데이터는 사용자의 로컬 브라우저에 저장되므로, 사용자가 확장 프로그램을 삭제하면 관련 설정 데이터도 즉시 함께 파기됩니다.

6. 이용자의 권리 및 행사 방법

이용자는 언제든지 확장 프로그램의 설정을 변경하거나 삭제함으로써 데이터 처리를 거부할 수 있습니다.
