# Flip the Shell 🐚

**Flip the Shell**은 웹 페이지 내에 숨겨진 다양한 쉘(Shell)과 민감한 엔드포인트를 탐지하고 분석하는 강력한 크롬 확장 프로그램입니다. 보안 연구원 및 개발자가 웹 애플리케이션의 구조를 빠르게 파악하고 잠재적인 취약점을 진단할 수 있도록 돕습니다.

## 🚀 주요 기능

- **자동 쉘 탐지:** 페이지 로드 시 백그라운드에서 자동으로 쉘 스크립트 및 관련 엔드포인트를 스캔합니다.
- **CORS 우회 지원:** 보안 제약이 있는 환경에서도 유연하게 데이터를 탐지할 수 있는 로직이 포함되어 있습니다.
- **실시간 알림:** 탐지된 요소들을 아이콘 배지 및 팝업을 통해 즉각적으로 알립니다.
- **사용자 친화적 인터페이스:** 복잡한 탐지 결과를 한눈에 확인할 수 있는 깔끔한 UI를 제공합니다.
- **K-Auto-Scan:** 효율적인 스캔 알고리즘을 통해 브라우저 성능 저하 없이 탐지 성능을 극대화합니다.

## 🛠 설치 방법

### 크롬 웹스토어에서 설치
아래 공식 링크를 통해 간편하게 브라우저에 추가할 수 있습니다:
- **[Flip the Shell - Chrome Web Store](https://chromewebstore.google.com/detail/flip-the-shell/opplmeompodbeojbbccnllpbjhcbnnbn?hl=ko&utm_source=ext_sidebar)**

### 개발자 모드로 설치 (로컬)
1. 이 저장소를 클론하거나 ZIP 파일로 다운로드합니다.
2. 크롬 브라우저에서 `chrome://extensions/`에 접속합니다.
3. 우측 상단의 **'개발자 모드'**를 활성화합니다.
4. **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭하고 프로젝트 폴더를 선택합니다.

## 💻 기술 스택

- **JavaScript (Vanilla):** 가볍고 빠른 실행 속도
- **CSS3:** 반응형 팝업 및 UI 디자인
